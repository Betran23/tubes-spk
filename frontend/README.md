# Frontend VIKOR

Frontend ini menggunakan React + Vite dengan gaya UI neo brutalism.

## Cara Menjalankan

1. Pastikan backend berjalan di `http://localhost:8000`.

2. Masuk ke folder frontend:

```bash
cd frontend
```

3. Install dependency:

```bash
npm install
```

4. Jalankan dev server:

```bash
npm run dev
```

5. Buka frontend:

```text
http://localhost:5173
```

Jika backend memakai URL lain, buat file `.env` di folder `frontend/` dan isi:

```text
VITE_API_BASE_URL=http://localhost:8000/api
```
